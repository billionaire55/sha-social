# SHA Social Auto-Post

Daily branded offer posts, generated by Claude and published via Postproxy.

**FIRST RUN IS A TEST**: `offers.json` in this package has every day's
`platforms` set to `["facebook"]` only, so the first manual run only posts to
Facebook no matter what day it is. Once that succeeds, replace `offers.json`
with `offers-FULL-RESTORE.json` (included separately, upload it too and rename
it to `offers.json`, overwriting this one) to restore the full 5-platform
weekly rotation.

Setup: see GO_LIVE_CHECKLIST.txt.
